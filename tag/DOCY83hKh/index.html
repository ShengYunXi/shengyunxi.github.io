我认识这么一个男孩
2022-06-28 4 min read # 故事

我认识这么一个男孩。

他曾对我说过自己对于跟前任的分开特别后悔不已，说自己当时真的不沉稳。

觉得恋爱谈到后期女孩儿总是事事都过分细致，是很难招架的捆绑。“我有段时间做了小手术，不能吃辣，她怕我难受，餐餐陪我吃清粥小菜，我说没必要的，她也不听。当时我觉得窒息。”

他提分手后的前几个月，真的松了一口气。又可以不被人管着，“啊你赶紧起床吃饭”这种。

但在那之后，他参加了保研的夏令营。激烈的竞争焦虑让人快速成长，他在那个夏天经常失眠，反思自己大学前几年的学习生涯，顺便也想起很多恋爱的小事。

这种时期总是很能让人成长，而人成长过后再回过头去看以前的亲密关系，总是会有截然不同的领悟。

别要睁开双眼
2022-06-24 2 min read # 故事

《相爱很难》的最后一句：别要睁开双眼。

想到死心塌地的在一起过，厌烦过，有时觉得可有可无，开始怀念开始的时光。

但若世上所有恋爱都只有开始，怎不会是另一种倦。何为“倦怠中的爱”？最初是眼皮打架困顿不已还要强撑着陪你说一会话，最尾是说不出来话的时候也要四目相对凝视一会。是一切刺激新鲜褪去换来的“伴侣之爱”。

既然说相爱是自然定理，无论和谁到最尾都是倦，那为何不是你陪我松懈这一生光阴。

唯独恋爱，得慢慢来
2022-06-22 2 min read # 故事

我觉得千万不要太快开始一段恋爱。

这个太快是指，如果你想谈一场普罗认知中的健康的正常的恋爱，就不要轻易对一个人产生“可以尝试一下”的念头。

不仅仅只是看到他的外貌，你还应该去和他聊天，和他散步，和他互相了解。听听他如何看待路边的猫，如何看待秋天的落叶，如何看待芋头奶茶，如何看待你脑海中浮现过的和他恋爱后会遇到的各种事情。

让他知道你讨厌吃香菜，你不适合穿橙色衣服，你喜欢被牵着手过马路，你看过好多遍的电影再看还是会哭……他喜欢的也不该仅仅是你，而是存在于生活里每个片段真实的你，或许有点自私有点小脾气又心软爱掉泪，但这才是他该了解的完全的你。

爱情说到底就是两个孤独的人
2022-06-20 3 min read # 故事 # 感悟

我看到过这样一段话：年轻的时候，一般有两种爱情，一种是那种高烧100°C的高温爱情，爱他爱到不行，发高烧。还有一种是长期低烧的爱情，就是你老是在想他到底爱不爱你，你感觉不到他的爱意，就这么赤眉红眼地耗着，耗到你筋疲力尽。

能给女孩子幸福的爱情，不是高烧也不是低烧，而是这种36.8℃的爱情，恒温，平和。幸福是什么？不需要你去燃烧自己，也不需要你低烧，真正的幸福不需要炫耀，幸福，就是偷着乐。

其实，保持恒温说简单也不简单，说难也不难，用最真实的自己和对方交往，并把这种真实和自然变成两个人相处的习惯就好。

我晚上经常饿，每次半夜起床室友都没睡。

既然都没睡，就唠嗑了几句。在我看来，她的喜怒哀乐全都源自于爱情。

我问她“吵架了吗？” “没吵架，只是太失望了。”我看到她的眼睛，我也真心感受到是内心深处涌上的难过。她说，大概就是，在同一座城市，却经常觉得自己在异地恋。

我明白室友的意思，我想女孩子都明白。

“他大概都不知道我打电话给他的意思就是希望他可以来到我身边，为什么他一点点都感受不到。”

你有没有很执着的喜欢过一个人？
2022-06-16 3 min read # 故事

你有没有很执着的喜欢过一个人？

数不清一天要想他多少次，也算不出为了他难过了多少回，明知道这样坚持下去的结果只会换来一次又一次的失望，却还是不肯放弃。
 
之所以不愿意放手，是因为两人之间有过太多美好的回忆，那些寂寞的时光，你们彼此取暖，互相鼓励着向前。那些看不到光的日子，他就像是一道暖阳笼罩着你，带给你希望和温暖。
 
他的存在，让你在平淡的生活里突然找到了意义，因为他，你开始变得柔软感性，也因为他，你慢慢懂得了爱。
 
你原本以为自己可以永远都陪伴在他的身边，目光随他而转，心随他而起伏，就仿佛整个世界都只剩下你和他。

我看到这么一个帖子。

女孩儿说，像做梦一样，自己跟认识不到半年的相亲对象在计划结婚了，事情简单得超乎想象。

下面有评论提醒她说，时间太短了，两个人只是吃吃饭逛逛街，真正的性格可能还没暴露出来，以后相处有风险的。

发帖的女孩儿回应说，哎呀，大家不用为我担心，我感觉他人品可以的，至于喜欢不喜欢的，我就不是那种“一定要跟喜欢的人结婚”的人，我一直觉得两个人能正常相处就可以结婚了。

我已经很久没有对一个人怦然心动过了
2022-01-14 2 min read # 故事 # 感悟

我已经很久没有对一个人怦然心动过了。

就像床底下结满的厚厚的灰尘，一层一层的覆盖上去，那一块地方永远是不被看见和不被打扫的。

也不知道从什么时候起，我看山不是山，看水不是水，就连看天，也跟以前看到的不同。我好怀念当时我们一起逃课，躺在学校操场上看飞机飞过天际，还从天空里拉出一道辉煌的长线的那时候。

你还记得那天的温度吗？我把手放在你心口，跟你说等你不想我的时候，你的心脏就不能再跳动了。你说我是个骗子，你要用你一辈子来佐证我说的不对。

然后，你吻了一下我的脸颊，你又不好意思的脸红了。那是我们很多很多个藏在试卷和习题册里的小甜蜜，伴随着书角的变黄和破旧，消失在那年夏天的风里。

我很容易陷入想证明对方到底爱不爱我的怪圈
2022-01-12 2 min read # 故事

我很容易陷入想证明对方到底爱不爱我的怪圈。

很奇怪，有时候是能感受到来自对方喜欢的，可总质疑“他是真的喜欢我吗”“他会喜欢我多久”“他会不会已经开始不喜欢我了”。

接着就完全乱套，他爱我的表现我很难再看到，眼里全是那些没有让我感到被爱的细枝末节。但你知道的，大家都是普通人，怎么可能会有人能把每个细节做到位。

带着所谓不被爱的证据，便开始猜忌和自怨自艾，把一件件小事放大到无可救药的地步，开始在心里谋划分手。矛盾愈演愈烈，感情在不自觉中朝着自己“希望”的方向发展，最后两个人身心疲惫，选择分开。

我认识一个女孩超级没有安全感
2022-01-10 3 min read # 故事 # 情感

我认识一个女孩超级没有安全感。

就连跟女性朋友相处，也常常会为了我是不是你最好的朋友或者我约了你三次你却没找过我一回这样无厘头的小事耍性子闹别扭。
　　
她谈过几场恋爱，几乎无一例外都因自己的小题大做作分手。而这些频繁的失去，又加重了她的疑心，即便是聚会的时候大家聊到一个她不了解的话题，她也要落泪：你们是不是都不喜欢我了，我是不是多余的。
　　
怎么说呢，女孩子过了20岁，这样的性格总是不够讨喜的，再浓厚的情谊，也抵不过一次又一次的抱怨和怀疑。后来聚会的时候，大家就开始有意无意地避开她。就这样了吧，所有人都以为这就是她的一生。
